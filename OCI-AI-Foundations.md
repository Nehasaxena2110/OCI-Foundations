# Oracle Cloud Infrastructure (OCI) AI Foundations

## What is OCI AI?

Oracle Cloud Infrastructure (OCI) AI provides a suite of AI and Machine Learning services that enable organizations to build, deploy, and manage intelligent applications. It offers both pre-trained AI services and infrastructure for training custom AI models.

---

# Artificial Intelligence (AI)

Artificial Intelligence (AI) enables machines to perform tasks that typically require human intelligence, such as reasoning, learning, decision-making, and problem-solving.

Examples:
- Virtual Assistants
- Chatbots
- Recommendation Systems
- Image Recognition
- Fraud Detection

---

# Machine Learning (ML)

Machine Learning is a subset of AI where systems learn from historical data and improve predictions without being explicitly programmed.

Types of ML:
- Supervised Learning
- Unsupervised Learning
- Reinforcement Learning

Examples:
- Spam Detection
- Price Prediction
- Customer Segmentation

---

# Deep Learning

Deep Learning uses Artificial Neural Networks with multiple hidden layers to solve complex problems.

Applications:
- Image Classification
- Speech Recognition
- Natural Language Processing (NLP)
- Autonomous Vehicles

---

# Generative AI

Generative AI creates new content based on user prompts.

Examples:
- Text Generation
- Image Generation
- Code Generation
- Audio Generation

Popular Models:
- GPT
- Llama
- Stable Diffusion

---

# Large Language Models (LLMs)

LLMs are AI models trained on massive text datasets to understand and generate human language.

Capabilities:
- Answer Questions
- Summarize Documents
- Generate Code
- Translate Languages
- Create Content

Examples:
- Llama
- Cohere
- GPT

---

# Retrieval-Augmented Generation (RAG)

RAG improves AI responses by retrieving relevant information from external knowledge sources before generating an answer.

Benefits:
- More accurate responses
- Reduced hallucinations
- Uses up-to-date enterprise knowledge
